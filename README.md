# Verilog Playground

A structured collection of Verilog designs covering fundamental to intermediate digital design concepts, with a focus on clean RTL design, verification, and FPGA implementation.

## Overview

This repository serves as a personal playground for experimenting with digital logic design and hardware development using Verilog. It includes a wide range of modules along with self-checking testbenches and FPGA-ready implementations.

## Features

- **Combinational Logic**
  - Adders, multiplexers, encoders/decoders

- **Finite State Machines (FSMs)**
  - Moore and Mealy machine implementations
  - Practical examples such as controllers

- **Verification**
  - Simulation using Icarus Verilog and GTKWave

- **FPGA Implementations**
  - Synthesizable designs tested using open-source FPGA toolchain:
    - Yosys (synthesis)
    - nextpnr (place & route)
    - openFPGALoader (programming)
  - Hardware deployment and validation on FPGA boards

## Goals
- Build a strong foundation in RTL design and digital logic
- Practice writing modular and reusable Verilog code
- Gain experience with simulation, verification, and FPGA workflows
